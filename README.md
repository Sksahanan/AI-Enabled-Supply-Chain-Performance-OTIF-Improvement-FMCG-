# AI-Enabled-Supply-Chain-Performance-OTIF-Improvement-FMCG-
Implemented an AI-driven analytics tool to monitor and improve supply chain KPIs for a consumer goods business. Analyzed logistics data to identify bottlenecks and predict delays, leading to a ~10% increase in On-Time-In-Full (OTIF) deliveries and more proactive issue resolution.
